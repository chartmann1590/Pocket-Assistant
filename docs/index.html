Capabilities

Built for real work

OCR, semantic search, optional self-hosted LLMs, and reminders — without sending your documents to a vendor backend.

📷

Smart capture

ML Kit text recognition for images; PDF text up to five pages. Import from share sheet or camera.

🧠

On-device AI

Download LiteRT-LM models (Qwen, Gemma 4, and more). Works offline after install.

🔍

Semantic search

Natural-language queries across saved items with on-device embeddings.

✓

Tasks & reminders

Extracted tasks with due dates and local notifications — your data stays on device.

☁️

Optional Ollama

Connect to your own server for heavier models; AUTO mode falls back to local when needed.

🔒

No telemetry

No cloud backend for core features. You control network use and remote endpoints.
